# Linux_Socket_Chatting
POSIX socket을 이용해 채팅을 할 수 있는 프로그램입니다

# How to use
### server
`$./server [port]`
해당 포트번호로 서버를 개방합니다

### client
`$./client [server-ip] [server-port] [nickname]`
고유의 닉네임으로 서버에 연결합니다
